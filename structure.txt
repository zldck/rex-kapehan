KEY FEATURES:
- User booking flow: app/page.js ? /api/bookings/hold ? /api/bookings/cleanup
- Admin dashboard: app/admin/page.js with real-time updates
- Reschedule: POST /api/admin/bookings/reschedule with email notifications
- Authentication: JWT tokens + magic links via Resend API
- Database: Supabase (PostgreSQL) with real-time queries
- Notifications: Email via Resend + Sound via HTML5 Audio (/public/sound/ding.mp3)
- Auto-refresh: 5-second polling on user calendar when enabled
- Slot cleanup: beforeunload handler releases pending slots on page close/refresh
